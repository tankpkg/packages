# Role-Specific CV Templates

Sources: CVCraft (2026), SoundCV (2026), Resumly (2026), GeniusResume (2026)

Covers: Tailored approaches for different career stages and transitions in tech.

## 1. Junior Engineer (0-2 Years)

### Profile

- Limited traditional experience
- Strong projects/internships
- Education relevant
- Often bootcamp or new grad

### Section Priority

```
1. Projects (show skills in action)
2. Technical Skills (explicit keyword list)
3. Education (relevant coursework)
4. Experience (internships, part-time)
5. Certifications (recent, relevant)
```

### Key Differences from Senior

- Projects section first (before experience)
- Include relevant coursework
- Class projects as evidence
- Open source contributions
- Hackathons/competitions

### Example: Junior Full-Stack

```
PROJECTS
---------
E-Commerce Platform | GitHub
- Built full-stack app with React, Node.js, PostgreSQL
- Implemented auth, cart, checkout (500+ users)
- Tech: React, Node.js, Stripe API, Docker

SKILLS
-------
Languages: JavaScript, Python, SQL
Frontend: React, CSS, HTML
Backend: Node.js, Express, Django
Tools: Git, Docker, AWS

EXPERIENCE
-----------
Web Development Intern | Company
- Built responsive UI components reducing load time by 30%
- Collaborated with 4-person team using Agile

EDUCATION
---------
BS Computer Science | University
Relevant: Data Structures, Web Development, Databases
```

## 2. Mid-Level Engineer (3-5 Years)

### Profile

- Solid work experience
- Technical depth in specific area
- Growing leadership
- Project ownership

### Section Priority

```
1. Experience (detailed, impact-focused)
2. Technical Skills
3. Projects (if significantly different from work)
4. Leadership (even small-team)
5. Education
```

### Key Content

- Promotion trajectory visible
- Technical ownership
- Cross-team collaboration
- Mentorship (even informal)
- Quantified impact

### Example: Mid-Level Backend

```
EXPERIENCE
----------
Software Engineer | Company (2022-Present)
- Designed and built microservices handling 1M+ daily requests
- Reduced API latency by 40% through caching strategy
- Led code review process for team of 6 engineers

Software Engineer | Company (2020-2022)
- Built Python API serving internal tools (10K+ users)
- Automated deployment reducing release time by 60%

SKILLS
------
Languages: Python, Go, JavaScript
Backend: FastAPI, Django, PostgreSQL, Redis
Cloud: AWS (EC2, Lambda, RDS)
```

## 3. Senior Engineer (5+ Years)

### Profile

- Deep technical expertise
- Significant leadership
- Architecture decisions
- Cross-team influence

### Section Priority

```
1. Experience (leadership and scale focus)
2. Technical Skills (breadth)
3. Leadership (team, hiring, direction)
4. Architecture/projects
5. Education
```

### Key Content

- Team size and scope
- Budget/technical ownership
- Cross-functional leadership
- Technical vision
- Hiring/institution building

### Example: Senior Backend

```
EXPERIENCE
----------
Senior Software Engineer | Company (2021-Present)
- Led backend architecture for platform serving 10M+ users
- Managed team of 5 engineers, conducting hiring and mentorship
- Designed migration from monolith to microservices, reduced costs 35%

Staff Software Engineer | Company (2019-2021)
- Built real-time data pipelines processing 50K events/second
- Established team coding standards and code review process
- Mentored 3 engineers, all promoted within 18 months

TECHNICAL LEADERSHIP
-------------------
- Technical design doc reviewer for team
- Platform architecture council member
- Speaker at internal engineering conference
```

## 4. Career Pivot: Dev to ML

### Profile

- Moving from software to ML
- Transferable skills
- Need to show ML readiness

### Section Priority

```
1. ML Projects (visible ML work)
2. Skills (new + transferable)
3. Learning (courses, certifications)
4. Experience (how skills transfer)
5. Education
```

### Pivot Narrative

**In summary:** "7 years software engineering with transition into ML through..."

**In experience:** Show how current role used data/automation

### Example: Dev → ML Engineer

```
PROJECTS (ML Focus)
------------------
Image Classification Model
- Built CNN classifier achieving 94% accuracy
- Used PyTorch, trained on 50K images
- Deployed model with FastAPI, 100 predictions/day

SKILLS (New + Transferable)
-------------------------
ML: Python, PyTorch, scikit-learn, TensorFlow
Data: Pandas, NumPy, SQL, feature engineering
Classical: Java, Go, REST APIs, Docker

ML COURSEWORK
-------------
- Coursera Deep Learning Specialization (2024)
- Fast.ai Practical Deep Learning (2023)
- Kaggle competitions: 2 bronze medals

EXPERIENCE
---------
Software Engineer | Company
- Built data processing pipelines for analytics
- Automated model training scripts in Python
```

## 5. Career Pivot: Non-Tech → Tech

### Profile

- No CS background
- Transfer from other industry
- Self-taught or bootcamp

### Section Priority

```
1. Projects (proof of ability)
2. Technical Skills
3. Relevant Experience (transferable)
4. Education (if relevant)
5. Side note on transition
```

### Narrative

- Don't hide non-tech background
- Frame transferable skills
- Show intentional transition
- Highlight self-learning

### Example: Consulting → SWE

```
PROJECTS
--------
E-Commerce Platform | Full-Stack Bootcamp
- Built with React, Node.js, PostgreSQL
- 3-month capstone project

SKILLS
------
Languages: JavaScript, Python, SQL
Frontend: React, HTML, CSS
Backend: Node.js, Express
Tools: Git, Docker

EXPERIENCE
----------
Senior Consultant | Consulting Firm
- Analyzed complex data sets for Fortune 500 clients
- Led teams of 5-10 on project implementations
- Developed analytical models using Excel/Python
- Transferable: Data analysis, project management, client communication
```

## Quick Decision Matrix

| Situation | Section Order |
|-----------|---------------|
| New grad / Junior | Projects → Skills → Education → Experience |
| Mid-level | Experience → Skills → Projects |
| Senior | Experience (leadership) → Skills → Architecture |
| Career pivot | Projects → Skills → Transferable experience |
| Career pivot (senior) | Experience → Projects → Skills |

## Tailoring per Application

### For Each Job, Adjust:

1. **Skills section** — Prioritize keywords from job description
2. **Projects shown** — Match role requirements
3. **Bullet emphasis** — Feature what aligns most
4. **Summary** — Mention role name
5. **Order** — Move relevant section up

### One-Click Tailoring

```
For job posting "Senior Backend Engineer, Python, Django":
1. Move Python to first in skills
2. Add Django (or ensure present)
3. Move backend projects up
4. Highlight microservices experience
5. Change summary to "Senior Backend Engineer"
```
